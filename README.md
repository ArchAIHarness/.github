# ArchAIHarness GitHub Profile

ArchAIHarness 组织门面与社区配置仓库。

## 定位

本仓库维护 ArchAIHarness 在 GitHub 上的组织首页、社区模板和公开协作口径。
其中 `profile/README.md` 是组织首页的核心内容。

## 目录

| 路径 | 说明 |
|---|---|
| `profile/README.md` | GitHub 组织首页内容 |
| `.github/ISSUE_TEMPLATE/` | Issue 模板与社区协作入口 |

## 边界

- 本仓库是公开资产，可以表达理念、方法、工具和抽象案例。
- 不提交客户材料、内部系统信息、未公开指标、真实密钥、Token、Cookie、`.env` 或账号密码。
- 对外表达应保守、专业、可验证，避免夸大战绩。
