# Icon: agent-master — Agent 调度控制面

## Design Story
Kubernetes 集群作为调度大脑，将容器化的 Agent 实例分发到每个用户。每个人获得一个专属的 Agent 运行环境。

## Style
Hand-drawn educational framework diagram style on warm cream paper background (#F5F0E8). Black sketch lines (#2C2C2C), soft pastel color blocks: #E8A87C warm orange, #85BAA1 sage green, #7BA7C9 soft blue. Handwritten Chinese text labels. Simple line art icons. No dark backgrounds, no 3D, no realistic human faces.

## Composition
Square 1:1. Left side: a K8s-style cube (调度器/Scheduler) with container boxes inside. Center: arrows flowing from the cube to the right. Right side: 3 simplified person figures, each receiving a small container/agent box. The container boxes have a small sparkle icon inside to indicate "agent". Warm orange (#E8A87C) accent on the K8s cube. Sage green (#85BAA1) accent on the agent boxes. Clean, minimal, readable at small size.
